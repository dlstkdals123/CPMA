## Exercise 04.12
Show the output produced by each of the following program fragments. Assume that ```i``` and ```j``` are ```int``` variables.
```
(a) i = 5;
    j = ++i * 3 - 2;
    printf("%d %d", i, j);
(b) i = 5;
    j = 3 - 2 * i++;
    printf("%d %d", i, j);
(c) i = 7;
    j = 3 * i-- + 2;
    printf("%d %d", i, j);
(d) i = 7;
    j = 3 + --i * 2;
    printf("%d %d", i, j);
```

## Answer
(a)   
My answer : 6 16   
Actual answer : 6 16   

(b)   
My answer : 6 -7   
Actual answer : 6 -7   

(c)   
My answer : 6 23   
Actual answer : 6 23   

(d)   
My answer : 6 15   
Actual answer : 6 15
